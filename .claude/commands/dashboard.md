# Дашборд вакансий

Запускает интерактивный терминальный дашборд.

## Инструкция

Выполни команду:

```bash
python -m src.ui.dashboard
```

## Управление дашбордом

| Клавиша | Действие |
|---------|----------|
| `1`–`N` | Выбрать вакансию по номеру |
| `j` / `k` | Вниз / вверх по списку |
| `n` / `p` | Следующая / предыдущая страница |
| `o` | Открыть вакансию в браузере |
| `a` | Откликнуться (с подтверждением) |
| `f` | Переключить фильтр грейда (A→B→C→D→F) |
| `r` | Обновить данные из БД |
| `q` | Выход |

## Пример использования

```
/dashboard
```
