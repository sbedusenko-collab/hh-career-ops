# Поиск и оценка вакансий

Запусти поиск и оценку вакансий на hh.ru по запросу пользователя.

## Инструкция

1. Прочитай `data/profile.yaml` — это профиль соискателя.
2. Прочитай `config.yaml` — параметры поиска (регион, зарплата, формат).
3. Запусти Python-скрипт:

```python
from src.core.evaluator import BatchEvaluator
evaluator = BatchEvaluator()
results = evaluator.run("$ARGUMENTS")
```

4. Покажи результаты в виде таблицы:

| Оценка | Грейд | Компания | Должность | Зарплата | Ссылка |
|--------|-------|----------|-----------|----------|--------|

5. Выдели ТОП-3 с кратким объяснением почему стоит откликнуться.
6. Предупреди о стоп-словах и dealbreaker вакансиях.

## Пример использования

```
/search Python разработчик
/search backend engineer удалённо
/search senior golang developer
```
