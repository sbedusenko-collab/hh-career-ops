# Анализ рыночных зарплат

Собирает вакансии по запросу и строит зарплатный отчёт: медиана, перцентили,
разбивка по опыту и формату работы, топ компаний.

## Инструкция

1. Запусти анализ:

```python
from src.core.salary_analysis import SalaryAnalyzer
from src.ui.salary_report import render_report

analyzer = SalaryAnalyzer()
report = analyzer.analyze("$ARGUMENTS")
render_report(report)
```

2. Покажи отчёт пользователю в формате:

```
══════════════════════════════════════════════
  Анализ зарплат: «Python разработчик»
══════════════════════════════════════════════

  Общая статистика (net, руб.):
  ┌─────────────────────────────────────────┐
  │ Всего вакансий:  312   С зарплатой: 187 │
  │ Минимум:      60 000   Максимум: 650 000 │
  │ 25-й перц.:  130 000   75-й перц.: 280 000│
  │ Среднее:     195 000   Медиана:  180 000 │
  └─────────────────────────────────────────┘

  Распределение: [гистограмма]

  По опыту:        По формату:
  Без опыта 90k    Удалённо  220k
  1–3 года  150k   Гибкий    180k
  3–6 лет   220k   Офис      160k
  6+ лет    300k

  Топ компаний по медиане...
```

3. Предложи:
   - `/salary <запрос> --trend` — динамика по дням (если есть история)
   - `/search <запрос>` — найти подходящие вакансии

## Примеры использования

```
/salary Python разработчик
/salary Senior Golang backend
/salary Data Scientist Москва
/salary Python разработчик --trend
```
